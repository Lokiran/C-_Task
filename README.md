
Program.cs – Student Program :

This file contains the main logic for the Student Information Program.  
It takes Name, Age, and Marks from the user, calculates the grade using if‑else conditions, and displays the result.  
It also supports entering multiple students using a loop and stores them in a list.


Fibonacci.cs – Fibonacci Series Class :

This file contains a class with a method that prints the Fibonacci sequence.  
It accepts a number from the user and generates that many Fibonacci values using a simple loop.  
Used to practice loops, variables, and method calling.


BonusTask.cs – Additional Practice Code :

This file includes extra learning code created for practicing C# fundamentals.  
It contains examples of user input handling, loops, lists, and basic operations.  
Useful for understanding how separate tasks can be organized within a project.
